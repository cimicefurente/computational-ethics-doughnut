# Doughnut Economics as an ASP Constraint Satisfaction Problem

Computational Ethics course project. Full write-up in
[`report/report.md`](report/report.md).

## What this is

Models Kate Raworth's Doughnut Economics (a social floor + an ecological
ceiling) as a constraint satisfaction problem in Answer Set Programming
(`clingo`), using real country data from O'Neill, Fanning, Lamb &
Steinberger (2018). Given a country and a menu of 14 policy levers, it
searches for the most parsimonious combination that satisfies every floor
and ceiling constraint at once; if none exists, it computes the minimal
remaining violation and identifies which dimensions are structurally out
of reach.

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Reproduce everything

```bash
# 1. Convert the O'Neill et al. (2018) spreadsheet into ASP facts
python3 src/data_prep.py

# 2. Check that lever effect sizes are representative of real cross-country variation
python3 src/representativeness_check.py

# 3. Run the three case studies
python3 src/run_scenario.py vietnam
python3 src/run_scenario.py costa_rica
python3 src/run_scenario.py united_states
```

## Layout

- `data/raw/` — the official Nature Sustainability supplementary spreadsheet.
- `data/processed/` — ASP facts and JSON lookups generated by `data_prep.py`.
- `src/doughnut.lp` — main feasibility encoding (hard floor/ceiling constraints).
- `src/best_effort.lp` — diagnostic encoding used when a country is UNSAT.
- `src/levers.lp` — the 14 policy levers and their assumed effects.
- `src/run_scenario.py` — CLI to run a scenario end to end.
- `results/` — JSON output per country (regenerated by `run_scenario.py`).
- `report/` — the written report and the representativeness table.

## Data source

O'Neill, D. W., Fanning, A. L., Lamb, W. F., & Steinberger, J. K. (2018).
A good life for all within planetary boundaries. *Nature Sustainability*,
1, 88–95. Supplementary data file downloaded directly from
[nature.com](https://www.nature.com/articles/s41893-018-0021-4).
